# wikitree
Tree structure of Wikidata entities

# Project Title

## Family Trees

This tool can show family trees of notable people, a tree of ancestors or descendants.

## Install NODEJS app

git clone https://github.com/dataprick/wikitree

cd wikitree/

cd nodejs-app/

npm install

npm run dev
